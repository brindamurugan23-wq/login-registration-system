# Login Registration System
## 🌐 About This Project
This is a 4-page web application developed as part of my BSc CS with AI 
studies. The app allows users to log in, get welcomed, register their 
details, and receive a success confirmation.

## 📄 Pages
- **Login Page** — User enters their login credentials
- **Welcome Page** — Welcomes the user after login
- **Registration Page** — New users can fill and submit their details
- **Success Page** — Confirms successful registration

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run
1. Download or clone this repository
2. Open login.html in any browser
3. Fill in the login details
4. Navigate through all 4 pages
